## [1.0.0] - 2025-12-14

### Added

#### Framework
- Complete Terminal Velocity research framework architecture
- Six damage vectors: Atmospheric (A), Biosphere (B), Hydrological (C), Chemical (D), Depletion (E), Systemic (F)
- 52 default mechanism definitions across all vectors
- Mechanism dossier template with 7 core components
- Evidence grading system (Grades I-VI) adapted from clinical medicine

#### Tooling
- `tv_scaffold.py` - Full project scaffolding CLI tool
  - Generates 1,000+ directories and 2,000+ files
  - Supports selective vector building
  - Custom mechanism import via YAML
  - Dry-run preview mode
  - Git initialization with .gitignore

#### Documentation
- `TERMINAL_VELOCITY_BLUEPRINT.md` - Complete framework specification
- `EVIDENCE_GRADING.md` - Detailed evidence standards guide
- `QUICKSTART.md` - 10-minute getting started guide
- `CONTRIBUTING.md` - Contribution guidelines
- `CODE_OF_CONDUCT.md` - Community standards

#### Visual Identity
- "4" signature color palette

  - Tertiary: #46ff84 (Electric Green)
  - Accent: #f5f5f4 (Soft White)
