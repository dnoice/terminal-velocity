# Causal Chain: A-01 - Coal Combustion Chain

## Executive Summary

The coal combustion causal chain spans from geological deposits to global climate impacts through a sequence of extraction, processing, transport, combustion, and atmospheric dispersion. Each link presents intervention opportunities, but the chain's momentum derives from locked-in infrastructure and financial commitments averaging 40+ years.

## 2. Chain Link Analysis

### 2.1 Link 1: Extraction

| Attribute | Value |
|-----------|-------|
| **Input** | Geological coal deposit |
| **Output** | Raw coal, mine waste |
| **Duration** | Mine life: 20-50 years |
| **Key Actors** | Mining companies, workers, regulators |
| **Side Effects** | Methane release, groundwater contamination, subsidence |
| **Latency** | Immediate extraction â†’ immediate methane release |

**Critical Dependencies:**
- Mining permits and land access
- Capital investment ($500M-$5B per large mine)
- Labor force (7.4 million global miners)
- Equipment and energy supply

### 2.2 Link 2: Processing

| Attribute | Value |
|-----------|-------|
| **Input** | Raw coal with impurities |
| **Output** | Washed coal, coal fines, slurry |
| **Duration** | Hours to days |
| **Key Actors** | Processing facilities, water utilities |
| **Side Effects** | Water pollution, slurry impoundment risks |
| **Latency** | Continuous during operations |

**Critical Dependencies:**
- Water supply (1,000-2,000 gallons/ton)
- Waste storage capacity
- Regulatory oversight

### 2.3 Link 3: Transportation

| Attribute | Value |
|-----------|-------|
| **Input** | Processed coal at mine |
| **Output** | Coal at power plant/port |
| **Duration** | Days to weeks |
| **Key Actors** | Rail operators, shipping companies, port authorities |
| **Side Effects** | Diesel emissions, coal dust, infrastructure dependency |
| **Latency** | Continuous during transport |

**Critical Dependencies:**
- Rail infrastructure (dedicated coal lines)
- Port capacity (Richards Bay: 76 Mt/year; Newcastle: 160 Mt/year)
- Shipping availability
- Energy for transport

### 2.4 Link 4: Combustion (Primary Damage Node)

| Attribute | Value |
|-----------|-------|
| **Input** | Coal + oxygen + heat |
| **Output** | Electricity + flue gas + ash |
| **Duration** | Continuous operation |
| **Key Actors** | Utilities, grid operators, regulators |
| **Primary Outputs** | CO2: 2.4 kg/kg coal; PM2.5, SO2, NOx, Hg |
| **Latency** | Immediate combustion â†’ immediate emissions |

**Critical Dependencies:**
- Grid connection and dispatch priority
- Fuel supply contracts
- Pollution control equipment
- Ash disposal capacity

### 2.5 Link 5: Atmospheric Dispersion

| Attribute | Value |
|-----------|-------|
| **Input** | Stack emissions |
| **Output** | Diluted pollutants across geography |
| **Duration** | Hours (local) to decades (global CO2) |
| **Key Actors** | Atmospheric physics, weather patterns |
| **Behavior** | Non-linear dispersion, accumulation in sinks |
| **Latency** | PM2.5: hours-days; CO2: centuries |

### 2.6 Link 6: Impact Realization

| Impact Category | Latency | Persistence |
|-----------------|---------|-------------|
| Acute respiratory | Hours-days | Reversible |
| Chronic respiratory | Years | Progressive |
| Cardiovascular | Years-decades | Progressive |
| Climate forcing | Decades | Centuries |
| Mercury bioaccumulation | Years-decades | Generational |

## 5. Intervention Points

### 5.1 High-Leverage Intervention Points

| Intervention Point | Mechanism | Effectiveness | Feasibility |
