# Claim Traceability: A-01 - Coal Combustion Chain

## Evidence Grading System

Based on clinical medicine hierarchy, adapted for sustainability research:

| Grade | Description | Criteria |
|-------|-------------|----------|
| I | Systematic reviews, meta-analyses | Multiple high-quality studies synthesized |
| II | Peer-reviewed original research | Single robust study, transparent methodology |
| III | Official statistics (government, IGO) | Authoritative institutional data |
| IV | Industry/NGO reports | Reputable organizations with methodology |
| V | Expert consensus, professional judgment | Acknowledged experts, reasoned analysis |
| VI | Anecdotal, journalistic, unverified | Useful but unconfirmed |

## 12. Verification Notes

### Verification Process

1. **Primary source check**: Locate original source document
2. **Methodology review**: Assess data collection and analysis methods
3. **Cross-validation**: Compare with alternative sources where available
4. **Date verification**: Confirm data currency
5. **Scope alignment**: Ensure source scope matches claim scope
